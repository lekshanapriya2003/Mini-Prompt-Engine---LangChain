# Mini-Prompt-Engine---LangChain

- Built a simple prompt engineering system using LangChain  
- Replaced hardcoded prompts with reusable templates  
- Supports multiple inputs: topic, audience, tone  
- Includes different prompt styles: Teaching, Interview, Storytelling  
- Uses chat-based prompts with role-based structure  
- Added basic input validation for cleaner outputs  
- Demonstrates a modular approach to prompt design  

**Tech:** Python, LangChain
